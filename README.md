# Power-Plant-ANN
In this repository i have load, build, train and evaluate an ANN model on power plant dataset having # AT -- Temperature value # V -- vaccum # AP -- pressure # RH -- Humidity # PE -- produced energy as features. The  output as r^2 score came out to be 93.43%
